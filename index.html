Responsible for developing Web applications using Ruby on Rails. One of the applications is an online tool for planning and analyzing data and images generated by a portable water robot named ESP. After analyzed large dataset, an online tool will generate the visualizations which allow users to interact, filter and export. The technologies behind these applications are Ruby on Rails, JavaScript, and Bootstrap. We mainly use open-source libraries of JavaScript such as jQuery, underscore.js, slickgrid, backbone, d3, nvd3. Other technologies we have been using are Rsync, Drupal and Apache Solr.
